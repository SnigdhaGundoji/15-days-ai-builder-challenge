# 📅 Daily Planner AI

An AI-powered productivity assistant that creates structured daily schedules from a list of user tasks using OpenRouter API.

---

## 🚀 Features

- Generate daily schedules
- Prioritize tasks automatically
- Create time slots
- Productivity recommendations
- Word counter
- Clean Streamlit UI
- AI-powered planning

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
