# 🚀 AI Career Coach

An AI-powered career guidance application that analyzes a user's skills and career goals, then generates personalized career advice, skill gap analysis, learning roadmaps, and project recommendations using OpenRouter API.

---

## 🚀 Features

- Analyze current skills
- Identify missing skills
- Generate learning roadmaps
- Suggest projects
- Provide career guidance
- Download career plan
- Clean Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
