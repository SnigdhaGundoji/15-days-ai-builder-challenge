# 📧 AI Email Writer

An AI-powered application that generates professional emails based on a user's purpose and selected tone using OpenRouter API.

---

## 🚀 Features

- Generate professional emails
- Multiple email tones
- Automatic subject generation
- Greeting and closing creation
- Download generated email
- Clean Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

## 📂 Project Structure

```text
Day-11-AI-Email-Writer/
│
├── app.py
├── README.md
├── requirements.txt
├── .env
```

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
