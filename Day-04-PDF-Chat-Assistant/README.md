# 📄 AI PDF Chat Assistant

An AI-powered application that reads uploaded PDF files and answers user questions based on the document content.

---

## 🚀 Features

- Upload PDF files
- Extract PDF text
- Ask questions from documents
- AI-generated answers
- Streamlit UI
- PDF processing using PyPDF2

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- PyPDF2
- OpenRouter API

---

## 📂 Project Structure

```text
Day-04-PDF-Chat-Assistant/

```

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
