# 📝 AI Quiz Generator

An AI-powered application that generates multiple-choice quizzes based on a selected topic and difficulty level using OpenRouter API.

---

## 🚀 Features

- Generate quizzes instantly
- Multiple difficulty levels
- Multiple-choice questions
- Answer key generation
- Download generated quiz
- Clean Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
