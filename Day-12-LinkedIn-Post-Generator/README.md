# 🚀 AI LinkedIn Post Generator

An AI-powered application that generates professional LinkedIn posts based on achievements, learning experiences, and career milestones using OpenRouter API.

---

## 🚀 Features

- Generate LinkedIn posts
- Multiple writing tones
- Professional formatting
- Key learnings section
- Relevant hashtags
- Download generated posts

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

## 📂 Project Structure

```text
Day-12-LinkedIn-Post-Generator/
│
├── app.py
├── README.md
├── requirements.txt
├── .env
```

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
