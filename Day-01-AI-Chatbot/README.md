# 🤖 AI Chatbot

A conversational AI chatbot built using Python, Streamlit, and OpenRouter API.

## Features
- AI-generated responses
- Conversational chat interface
- Session memory
- Clear chat button
- AI personality system
- Sidebar UI

## Technologies Used
- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API
