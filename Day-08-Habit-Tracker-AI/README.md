# ✅ Habit Tracker AI

An AI-powered habit analysis application that helps users track, improve, and optimize their habits using OpenRouter API.

---

## 🚀 Features

- Habit categorization
- Tracking suggestions
- Consistency improvement tips
- Weekly improvement advice
- Priority habit identification
- Word counter
- Clean Streamlit UI

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

## 📂 Project Structure

```text
Day-08-Habit-Tracker-AI/
│

```

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
