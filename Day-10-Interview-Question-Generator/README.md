# 🎯 AI Interview Question Generator

An AI-powered application that generates interview questions, important concepts, and preparation tips based on a selected topic and difficulty level using OpenRouter API.

---

## 🚀 Features

- Generate interview questions
- Beginner, Intermediate, and Advanced levels
- Important concept suggestions
- Interview preparation tips
- Download generated questions
- Clean Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
