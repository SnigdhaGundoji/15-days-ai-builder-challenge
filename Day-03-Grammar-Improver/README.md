# ✍️ AI Grammar & English Improver

An AI-powered application that improves grammar, fluency, and communication using OpenRouter API and Streamlit.

---

## 🚀 Features

- Grammar correction
- Professional English rewriting
- Fluent English improvement
- Simple English rewriting
- Word counter
- Character counter
- Expandable response section
- Clean Streamlit UI

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

## 📂 Project Structure

```text
Day-03-Grammar-Improver/
│
├── app.py

```

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
