# 💡 Motivation Quote Generator

An AI-powered application that generates motivational quotes, practical advice, and action steps based on a user's situation using OpenRouter API.

---

## 🚀 Features

- Generate motivational quotes
- Personalized advice
- Actionable next steps
- Mood-based motivation
- Download generated motivation
- Clean Streamlit interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

## 📂 Project Structure

```text
Day-09-Motivation-Quote-Generator/
│
├── app.py

```

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
