# 📄 AI Resume Analyzer

An AI-powered application that analyzes resumes and provides professional feedback using OpenRouter API.

---

## 🚀 Features

- Upload Resume PDF
- Extract resume text
- Analyze strengths
- Identify weaknesses
- Suggest missing skills
- Provide improvement recommendations
- Resume score out of 10
- Suitable job role suggestions

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- PyPDF2
- OpenRouter API

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
