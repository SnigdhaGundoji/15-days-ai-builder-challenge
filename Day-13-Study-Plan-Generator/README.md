# 📚 AI Study Plan Generator

An AI-powered application that creates personalized study roadmaps based on a learning goal and duration using OpenRouter API.

---

## 🚀 Features

- Generate study plans
- Custom learning goals
- Flexible study duration
- Weekly roadmap creation
- Practice recommendations
- Final project suggestions
- Download study plan

---

## 🛠 Technologies Used

- Python
- Streamlit
- Requests
- Python-dotenv
- OpenRouter API

---

Day-13-Study-Plan-Generator/

---

## ▶️ Run The Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file:

```text
OPENROUTER_API_KEY=your_api_key
```
